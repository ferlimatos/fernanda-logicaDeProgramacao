# Exercício 5

# Explique a função do comando range(start, stop, step) e descreva o papel de cada
parâmetro e qual parâmetro é obrigatório.

===================================

Minha resposta:

O comando range(start, stop, step) gera uma sequência numérica. O start define onde começa (opcional), o stop define onde termina e é o único parâmetro obrigatório, e o step define o incremento entre os valores (opcional).

Exemplo: 

range(1,5,1) gera os números: 1, 2, 3, 4.
