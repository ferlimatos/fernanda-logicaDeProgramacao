# Lógica e Programação

Este repositório serve como um caderno digital e portfólio para os exercícios, projetos e conceitos aprendidos na disciplina de Lógica e Programação do curso técnico de Desenvolvimento Web e Mobile da [Escola do Futuro José Luiz Bittencourt](https://efg.org.br/goiania) em Goiânia. O foco principal é no desenvolvimento do pensamento lógico e na aplicação desses conceitos utilizando a linguagem Python.

## Mini-projetos
Estes projetos foram desenvolvidos para aplicar os conceitos de variáveis, estruturas de controle (if/elif/else, for, while), e listas na prática.

## Sobre a Disciplina
A disciplina abrange os fundamentos essenciais para qualquer desenvolvedor, independentemente da linguagem de programação. Os tópicos principais incluem:

- Algoritmos e Estruturas de Dados Básicas: A base para a resolução de problemas.

- Variáveis e Tipos de Dados
  - Variáveis: Nomes que referenciam valores armazenados na memória. Em Python, toda variável é tratada como um objeto.
  - Tipos Primitivos Básicos (em Python):
    - int: Números inteiros.
    - float: Números reais (ou flutuantes).
    - str: Sequências de caracteres (textos).
    - bool: Valores lógicos (True ou False).
    - Boas Práticas: Regras para nomeação de variáveis e a importância de nomes descritivos.

- Estruturas de Controle:
  - Comandos de Entrada e Saída: Utilização das funções print() e input().
  - Operadores Aritméticos: Adição, Subtração, Multiplicação, Divisão, Divisão Inteira, Módulo e Exponenciação.
  - Ordem de Precedência: A regra PEMDAS (Parênteses, Expoentes, Multiplicação, Divisão, Adição, Subtração).

  - Condicionais (Tomada de Decisão)
    - Condições Simples: Uso do comando if.
    - Condições Compostas: Uso dos comandos if e else.
    - Múltiplas Condições: Uso do trio if, elif e else para testar condições adicionais.
    - Condições Aninhadas: Aninhar uma condição dentro de outra, onde a execução de uma depende da anterior.
   
  - Repetição/Laços
    - Estrutura while: Repete um bloco de código enquanto uma condição for verdadeira (para um número desconhecido de vezes).
    - Estrutura for: Repete um bloco de código iterando sobre uma sequência, como um range de números ou uma lista.
    - Função range(): Utilizada para gerar sequências, com parâmetros start, stop e step.
    - Ferramentas de Controle: Conceitos de Contador e Acumulador em loops , e o uso do comando break para encerrar um laço prematuramente.

- Estruturas de Dados em Python: Listas
    - Introdução às Listas: Uma coleção fundamental para armazenar múltiplos valores em uma única variável.
    - Acessar e Modificar Elementos: Referenciar e alterar elementos usando a posição (índice).
    - Adicionar Elementos: Usando append() (adiciona ao final) e insert() (adiciona em posição específica).
    - Remover Elementos: Usando remove("valor") ou del lista[posição].
    - Ordenação: Utilizando .sort() e a função sorted().
    - Funções Auxiliares: Uso das funções max() e min().
